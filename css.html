Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages, including colors, fonts, spacing, and positioning. CSS works alongside HTML to create visually appealing and responsive designs.

Why Use CSS?

CSS is essential for web development because it allows you to:

  • Separate content (HTML) from presentation (CSS).
  • Apply consistent styles across multiple pages.
  • Create responsive designs for different devices.
  • Enhance user experience with animations and transitions.

Basic CSS Syntax

CSS consists of selectors and declarations. A selector targets HTML elements, and declarations define the styles to apply.

Example:

/* Selector */ h1 { /* Declarations */ color: blue; font-size: 24px; }

Common CSS Properties

Here are some commonly used CSS properties:

Example:

/* Text Properties */ h1 { color: red; font-family: Arial, sans-serif; font-size: 2rem; text-align: center; } /* Box Model Properties */ div { width: 100%; padding: 20px; margin: 10px; border: 1px solid #ccc; background-color: #f9f9f9; } /* Flexbox Layout */ .container { display: flex; justify-content: center; align-items: center; } /* Responsive Design */ @media (max-width: 768px) { body { font-size: 14px; } }

How to Use CSS

There are three ways to apply CSS to an HTML document:

  1. Inline CSS: Apply styles directly to HTML elements using the style attribute.
  2. Internal CSS: Define styles within a <style> tag in the <head> section.
  3. External CSS: Link to an external CSS file using the <link> tag.

Example:

<!-- Inline CSS --> <h1 style="color: blue;">Hello, World!</h1> <!-- Internal CSS --> <style> h1 { color: blue; } </style> <!-- External CSS --> <link rel="stylesheet" href="styles.css">

CSS Selectors

CSS selectors are used to target HTML elements for styling. Here are some common selectors:

Example:

/* Element Selector */ p { color: green; } /* Class Selector */ .highlight { background-color: yellow; } /* ID Selector */ #header { font-size: 2rem; } /* Attribute Selector */ a[target="_blank"] { color: purple; } /* Pseudo-class Selector */ a:hover { text-decoration: underline; }

CSS Box Model

The CSS box model describes the layout of elements, including content, padding, border, and margin.

Example:

div { width: 300px; padding: 20px; border: 5px solid black; margin: 10px; background-color: lightblue; }

CSS Flexbox

Flexbox is a layout model that allows you to design flexible and responsive layouts.

Example:

.container { display: flex; justify-content: space-between; align-items: center; } .item { flex: 1; margin: 10px; padding: 20px; background-color: #f4f4f4; }

CSS Grid

CSS Grid is a powerful layout system for creating complex, responsive designs.

Example:

.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; } .grid-item { background-color: #f9f9f9; padding: 20px; border: 1px solid #ccc; }

CSS Media Queries

Media queries allow you to apply styles based on the device's screen size, orientation, or other features.

Example:

@media (max-width: 768px) { body { font-size: 14px; } .container { flex-direction: column; } }

HTML Tags and CSS Selectors Reference

1. CSS Types (Ways to Apply CSS)

Different ways to apply CSS to HTML elements.

Inline CSS

CSS is applied directly to an HTML element using the style attribute.

Syntax:

<element style="property: value;">

Example:

<p style="color: red; font-size: 16px;">This is a paragraph with inline CSS.</p>

Output:

This is a paragraph with inline CSS.

Internal CSS

CSS is defined within a <style> tag in the <head> section of the HTML document.

Syntax:

<style> selector { property: value; } </style>

Example:

<style> p { color: blue; font-size: 18px; } </style>

Output:

This is a paragraph with internal CSS.

External CSS

CSS is defined in a separate .css file and linked to the HTML document using the <link> tag.

Syntax:

<link rel="stylesheet" href="styles.css">

Example:

styles.css:

p { color: green; font-size: 20px; }

HTML:

<link rel="stylesheet" href="styles.css"> <p>This is a paragraph with external CSS.</p>

Output:

This is a paragraph with external CSS.

2. CSS Selectors

Used to target HTML elements for styling.

Universal Selector (*)

Selects all elements on the page.

Syntax:

* { /* Styles apply to all elements */ }

Example:

* { margin: 0; padding: 0; }

Type Selector (p, div, h1)

Selects all elements of a specific type.

Syntax:

p { /* Styles apply to all <p> elements */ }

Example:

p { color: blue; font-size: 16px; }

Class Selector (.classname)

Selects all elements with a specific class.

Syntax:

.classname { /* Styles apply to all elements with class="classname" */ }

Example:

.highlight { background-color: yellow; }

ID Selector (#idname)

Selects a single element with a specific ID.

Syntax:

#idname { /* Styles apply to the element with id="idname" */ }

Example:

#header { font-size: 24px; color: red; }

Grouping Selector (h1, p, div)

Selects multiple elements and applies the same styles.

Syntax:

h1, p, div { /* Styles apply to <h1>, <p>, and <div> elements */ }

Example:

h1, p, div { margin-bottom: 10px; }

Combinator Selectors

Selects elements based on their relationship to other elements.

Syntax:

/* Descendant Selector */ div p { /* Styles apply to <p> elements inside <div> */ } /* Child Selector */ div > p { /* Styles apply to <p> elements that are direct children of <div> */ } /* Adjacent Sibling Selector */ h1 + p { /* Styles apply to <p> elements immediately following <h1> */ } /* General Sibling Selector */ h1 ~ p { /* Styles apply to all <p> elements that are siblings of <h1> */ }

Example:

/* Descendant Selector */ div p { color: green; } /* Child Selector */ div > p { font-weight: bold; } /* Adjacent Sibling Selector */ h1 + p { margin-top: 0; } /* General Sibling Selector */ h1 ~ p { font-style: italic; }

Attribute Selectors

Selects elements based on their attributes.

Syntax:

/* [attribute] */ [target] { /* Styles apply to elements with the "target" attribute */ } /* [attribute="value"] */ [type="submit"] { /* Styles apply to elements with type="submit" */ } /* [attribute^="value"] */ [href^="https"] { /* Styles apply to elements with href starting with "https" */ }

Example:

/* [attribute] */ [target] { border: 1px solid black; } /* [attribute="value"] */ [type="submit"] { background-color: green; } /* [attribute^="value"] */ [href^="https"] { color: blue; }

Pseudo-classes

Selects elements based on their state or position.

Syntax:

/* :hover */ a:hover { /* Styles apply when the link is hovered */ } /* :focus */ input:focus { /* Styles apply when the input is focused */ } /* :nth-child(n) */ li:nth-child(2) { /* Styles apply to the 2nd <li> element */ } /* :not() */ p:not(.special) { /* Styles apply to <p> elements without class="special" */ }

Example:

/* :hover */ a:hover { color: red; } /* :focus */ input:focus { border-color: blue; } /* :nth-child(n) */ li:nth-child(2) { font-weight: bold; } /* :not() */ p:not(.special) { font-size: 14px; }

Pseudo-elements

Selects specific parts of an element.

Syntax:

/* ::before */ p::before { content: "Before "; } /* ::after */ p::after { content: " After"; } /* ::first-letter */ p::first-letter { font-size: 24px; } /* ::first-line */ p::first-line { font-weight: bold; }

Example:

/* ::before */ p::before { content: "Before "; color: green; } /* ::after */ p::after { content: " After"; color: red; } /* ::first-letter */ p::first-letter { font-size: 24px; color: blue; } /* ::first-line */ p::first-line { font-weight: bold; }

3. CSS Properties (Used to Style Elements)

CSS properties are used to style HTML elements.

Box Model

Properties related to the box model.

Syntax:

/* Width and Height */ width: 200px; height: 100px; /* Margin and Padding */ margin: 10px; padding: 15px; /* Border and Border Radius */ border: 2px solid black; border-radius: 5px; /* Box Sizing */ box-sizing: border-box;

Example:

<div style="width: 200px; height: 100px; margin: 10px; padding: 15px; border: 2px solid black; border-radius: 5px; box-sizing: border-box;"> Box Model Example </div>

Output:

Box Model Example

Text & Fonts

Properties related to text and fonts.

Syntax:

/* Color and Background */ color: red; background-color: yellow; /* Font Family and Size */ font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; /* Text Alignment and Transformation */ text-align: center; text-transform: uppercase; /* Letter and Word Spacing */ letter-spacing: 2px; word-spacing: 5px; /* Text Decoration and Shadow */ text-decoration: underline; text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);

Example:

<p style="color: red; background-color: yellow; font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; text-align: center; text-transform: uppercase; letter-spacing: 2px; word-spacing: 5px; text-decoration: underline; text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);"> Text and Fonts Example </p>

Output:

Text and Fonts Example

Positioning & Layout

Properties related to positioning and layout.

Syntax:

/* Display */ display: flex; /* Position */ position: relative; top: 10px; left: 20px; /* Z-Index */ z-index: 1; /* Visibility */ visibility: visible; /* Overflow */ overflow: hidden;

Example:

<div style="position: relative; top: 10px; left: 20px; z-index: 1; visibility: visible; overflow: hidden;"> Positioning and Layout Example </div>

Output:

Positioning and Layout Example

Flexbox

Properties related to Flexbox layout.

Syntax:

/* Flex Container */ display: flex; flex-direction: row; justify-content: center; align-items: center; /* Flex Items */ flex-grow: 1; flex-shrink: 1;

Example:

<div style="display: flex; flex-direction: row; justify-content: center; align-items: center;"> <div style="flex-grow: 1; background-color: lightblue; padding: 10px;">Item 1</div> <div style="flex-grow: 1; background-color: lightcoral; padding: 10px;">Item 2</div> </div>

Output:

Item 1
Item 2

Grid Layout

Properties related to CSS Grid layout.

Syntax:

/* Grid Container */ display: grid; grid-template-columns: 1fr 1fr; grid-template-rows: auto; grid-gap: 10px; /* Grid Items */ grid-area: 1 / 1 / 2 / 2;

Example:

<div style="display: grid; grid-template-columns: 1fr 1fr; grid-gap: 10px;"> <div style="background-color: lightblue; padding: 10px;">Item 1</div> <div style="background-color: lightcoral; padding: 10px;">Item 2</div> </div>

Output:

Item 1
Item 2

Properties related to backgrounds and borders.

Syntax:

/* Background */ background: lightblue; background-image: url('image.jpg'); /* Border */ border: 2px solid black; border-radius: 5px; /* Box Shadow */ box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5); /* Opacity */ opacity: 0.8;

Example:

<div style="background: lightblue; border: 2px solid black; border-radius: 5px; box-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5); opacity: 0.8; padding: 10px;"> Background and Borders Example </div>

Background and Borders Example

Lists & Tables

Properties related to lists and tables.

Syntax:

/* List Style */ list-style-type: square; list-style-position: inside; /* Table Borders */ border-collapse: collapse; border-spacing: 0;

Example:

<ul style="list-style-type: square; list-style-position: inside;"> <li>Item 1</li> <li>Item 2</li> </ul> <table style="border-collapse: collapse; border-spacing: 0;"> <tr> <td style="border: 1px solid black; padding: 5px;">Cell 1</td> <td style="border: 1px solid black; padding: 5px;">Cell 2</td> </tr> </table>

Output:

  • Item 1
  • Item 2
Cell 1 Cell 2

Transitions & Animations

Properties related to transitions and animations.

Syntax:

/* Transition */ transition: background-color 0.5s ease; /* Animation */ animation: slide 2s infinite; /* Transform */ transform: rotate(45deg); /* Keyframes */ @keyframes slide { 0% { transform: translateX(0); } 100% { transform: translateX(100px); } }

Example:

<div style="width: 100px; height: 100px; background-color: lightblue; transition: background-color 0.5s ease;" onmouseover="this.style.backgroundColor='lightcoral'" onmouseout="this.style.backgroundColor='lightblue'"> Hover Me </div>

Output:

Hover Me

Media Queries & Responsive Design

Properties related to responsive design and media queries.

Syntax:

Example:

<style> @media (max-width: 768px) { .responsive-example { font-size: 14px; } } </style> <div class="responsive-example" style="width: 50vw; height: 50vh; font-size: 2rem; margin: 1em;"> Responsive Design Example </div>

Responsive Design Example
