Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages. HTML provides the structure of a webpage by using a system of tags and attributes to define elements like headings, paragraphs, links, images, and more.

Why Use HTML?

HTML is the backbone of all websites. It allows you to:

  • Create structured content for web pages.
  • Add text, images, videos, and other media.
  • Build forms for user input.
  • Link pages together using hyperlinks.
  • Make content accessible and SEO-friendly.

Basic HTML Structure

Every HTML document follows a basic structure. Here's an example:

Example:

<html> <head> <title> </title> </head> <body> <h1>Welcome to My Website</h1> <p>This is a paragraph of text.</p> </body> </html>

How to Use HTML

To create an HTML file:

  1. Open a text editor (e.g., Notepad, VS Code).
  2. Write your HTML code.
  3. Save the file with a .html extension (e.g., index.html).
  4. Open the file in a web browser to view your webpage.

Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages, including colors, fonts, spacing, and positioning. CSS works alongside HTML to create visually appealing and responsive designs.

Why Use CSS?

CSS is essential for web development because it allows you to:

  • Separate content (HTML) from presentation (CSS).
  • Apply consistent styles across multiple pages.
  • Create responsive designs for different devices.
  • Enhance user experience with animations and transitions.

Basic CSS Syntax

CSS consists of selectors and declarations. A selector targets HTML elements, and declarations define the styles to apply.

Example:

/* Selector */ h1 { /* Declarations */ color: blue; font-size: 24px; }

Common CSS Properties

Here are some commonly used CSS properties:

Example:

/* Text Properties */ h1 { color: red; font-family: Arial, sans-serif; font-size: 2rem; text-align: center; } /* Box Model Properties */ div { width: 100%; padding: 20px; margin: 10px; border: 1px solid #ccc; background-color: #f9f9f9; } /* Flexbox Layout */ .container { display: flex; justify-content: center; align-items: center; } /* Responsive Design */ @media (max-width: 768px) { body { font-size: 14px; } }

How to Use CSS

There are three ways to apply CSS to an HTML document:

  1. Inline CSS: Apply styles directly to HTML elements using the style attribute.
  2. Internal CSS: Define styles within a <style> tag in the <head> section.
  3. External CSS: Link to an external CSS file using the <link> tag.

Example:

<!-- Inline CSS --> <h1 style="color: blue;">Hello, World!</h1> <!-- Internal CSS --> <style> h1 { color: blue; } </style> <!-- External CSS --> <link rel="stylesheet" href="styles.css">

CSS Selectors

CSS selectors are used to target HTML elements for styling. Here are some common selectors:

Example:

/* Element Selector */ p { color: green; } /* Class Selector */ .highlight { background-color: yellow; } /* ID Selector */ #header { font-size: 2rem; } /* Attribute Selector */ a[target="_blank"] { color: purple; } /* Pseudo-class Selector */ a:hover { text-decoration: underline; }

CSS Box Model

The CSS box model describes the layout of elements, including content, padding, border, and margin.

Example:

div { width: 300px; padding: 20px; border: 5px solid black; margin: 10px; background-color: lightblue; }

CSS Flexbox

Flexbox is a layout model that allows you to design flexible and responsive layouts.

Example:

.container { display: flex; justify-content: space-between; align-items: center; } .item { flex: 1; margin: 10px; padding: 20px; background-color: #f4f4f4; }

CSS Grid

CSS Grid is a powerful layout system for creating complex, responsive designs.

Example:

.grid-container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 10px; } .grid-item { background-color: #f9f9f9; padding: 20px; border: 1px solid #ccc; }

CSS Media Queries

Media queries allow you to apply styles based on the device's screen size, orientation, or other features.

Example:

@media (max-width: 768px) { body { font-size: 14px; } .container { flex-direction: column; } }

HTML Tags and CSS Selectors Reference

1. CSS Types (Ways to Apply CSS)

Different ways to apply CSS to HTML elements.

Inline CSS

CSS is applied directly to an HTML element using the style attribute.

Syntax:

<element style="property: value;">

Example:

<p style="color: red; font-size: 16px;">This is a paragraph with inline CSS.</p>

Output:

This is a paragraph with inline CSS.

Internal CSS

CSS is defined within a <style> tag in the <head> section of the HTML document.

Syntax:

<style> selector { property: value; } </style>

Example:

<style> p { color: blue; font-size: 18px; } </style>

Output:

This is a paragraph with internal CSS.

External CSS

CSS is defined in a separate .css file and linked to the HTML document using the <link> tag.

Syntax:

<link rel="stylesheet" href="styles.css">

Example:

styles.css:

p { color: green; font-size: 20px; }

HTML:

<link rel="stylesheet" href="styles.css"> <p>This is a paragraph with external CSS.</p>

Output:

This is a paragraph with external CSS.

2. CSS Selectors

Used to target HTML elements for styling.

Universal Selector (*)

Selects all elements on the page.

Syntax:

* { /* Styles apply to all elements */ }

Example:

* { margin: 0; padding: 0; }

Type Selector (p, div, h1)

Selects all elements of a specific type.

Syntax:

p { /* Styles apply to all <p> elements */ }

Example:

p { color: blue; font-size: 16px; }

Class Selector (.classname)

Selects all elements with a specific class.

Syntax:

.classname { /* Styles apply to all elements with class="classname" */ }

Example:

.highlight { background-color: yellow; }

ID Selector (#idname)

Selects a single element with a specific ID.

Syntax:

#idname { /* Styles apply to the element with id="idname" */ }

Example:

#header { font-size: 24px; color: red; }

Grouping Selector (h1, p, div)

Selects multiple elements and applies the same styles.

Syntax:

h1, p, div { /* Styles apply to <h1>, <p>, and <div> elements */ }

Example:

h1, p, div { margin-bottom: 10px; }

Combinator Selectors

Selects elements based on their relationship to other elements.

Syntax:

/* Descendant Selector */ div p { /* Styles apply to <p> elements inside <div> */ } /* Child Selector */ div > p { /* Styles apply to <p> elements that are direct children of <div> */ } /* Adjacent Sibling Selector */ h1 + p { /* Styles apply to <p> elements immediately following <h1> */ } /* General Sibling Selector */ h1 ~ p { /* Styles apply to all <p> elements that are siblings of <h1> */ }

Example:

/* Descendant Selector */ div p { color: green; } /* Child Selector */ div > p { font-weight: bold; } /* Adjacent Sibling Selector */ h1 + p { margin-top: 0; } /* General Sibling Selector */ h1 ~ p { font-style: italic; }

Attribute Selectors

Selects elements based on their attributes.

Syntax:

/* [attribute] */ [target] { /* Styles apply to elements with the "target" attribute */ } /* [attribute="value"] */ [type="submit"] { /* Styles apply to elements with type="submit" */ } /* [attribute^="value"] */ [href^="https"] { /* Styles apply to elements with href starting with "https" */ }

Example:

Recursion is a technique where a function calls itself to solve a smaller version of a problem, until it reaches a base case.

  if (n === 0) return 1;
  return n * factorial(n - 1);

A Promise is an object that represents the eventual completion or failure of an asynchronous operation. Promises have three states: pending, fulfilled, and rejected.

Callback Hell occurs when callbacks are nested within callbacks, leading to code that is hard to read and maintain. This often happens in asynchronous code without proper structure.

To avoid callback hell, use Promises or async/await for better readability and maintainability.

JavaScript map(), filter(), and reduce()

These powerful array methods help you transform, filter, and summarize data efficiently. They are widely used in functional programming patterns.

1. map()

map() creates a new array by applying a function to each element in the original array.

const numbers = [1, 2, 3, 4]; const doubled = numbers.map(num => num * 2); console.log(doubled); // [2, 4, 6, 8]

2. filter()

filter() returns a new array with elements that pass a test (return true).

const numbers = [1, 2, 3, 4, 5]; const evens = numbers.filter(num => num % 2 === 0); console.log(evens); // [2, 4]

3. reduce()

reduce() reduces an array to a single value by applying a function to each element (accumulator pattern).

const numbers = [1, 2, 3, 4]; const sum = numbers.reduce((total, num) => total + num, 0); console.log(sum); // 10

4. When to Use Each

  • map(): When you want to transform items in an array.
  • filter(): When you want only certain items from an array.
  • reduce(): When you want a single value (sum, average, object, etc.).

5. Real-World Combo Example

Combining all three methods:

const products = [ { name: "Pen", price: 10 }, { name: "Book", price: 50 }, { name: "Pencil", price: 5 } ]; const total = products .filter(p => p.price > 5) .map(p => p.price) .reduce((sum, price) => sum + price, 0); console.log(total); // 60

6. Key Notes

  • These methods do not change the original array.
  • map() and filter() return arrays, reduce() returns a single value.
  • Can be chained together for powerful data manipulation.
