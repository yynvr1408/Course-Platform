Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages. HTML provides the structure of a webpage by using a system of tags and attributes to define elements like headings, paragraphs, links, images, and more.

Why Use HTML?

HTML is the backbone of all websites. It allows you to:

  • Create structured content for web pages.
  • Add text, images, videos, and other media.
  • Build forms for user input.
  • Link pages together using hyperlinks.
  • Make content accessible and SEO-friendly.

Basic HTML Structure

Every HTML document follows a basic structure. Here's an example:

Example:

<html> <head> <title> </title> </head> <body> <h1>Welcome to My Website</h1> <p>This is a paragraph of text.</p> </body> </html>

How to Use HTML

To create an HTML file:

  1. Open a text editor (e.g., Notepad, VS Code).
  2. Write your HTML code.
  3. Save the file with a .html extension (e.g., index.html).
  4. Open the file in a web browser to view your webpage.

<html>

The root element of an HTML document.

Syntax:

<html>...</html>

Example:

<html> <head>...</head> <body>...</body> </html>

Container for metadata (e.g., title, styles, scripts).

<head>...</head>

<head> <title>Page Title</title> </head>

<body>

Syntax:

Example:

2.Metadata in HTML

<meta charset="UTF-8">

Specifies the character encoding for the HTML document.

Syntax:

<meta charset="UTF-8">

Example:

<meta charset="UTF-8">

<meta name="viewport" content="...">

Controls the layout and scaling of the webpage on mobile devices.

Syntax:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Example:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<meta name="description" content="...">

Provides a brief description of the webpage for search engines.

Syntax:

<meta name="description" content="Description of the webpage">

Example:

<meta name="description" content="Learn about metadata in HTML.">

<meta name="keywords" content="...">

Specifies a list of keywords related to the webpage.

Syntax:

<meta name="keywords" content="keyword1, keyword2, keyword3">

Example:

<meta name="keywords" content="HTML, metadata, web development">

<meta name="author" content="...">

Specifies the author of the webpage.

Syntax:

<meta name="author" content="Author Name">

Example:

<meta name="author" content="John Doe">

<meta http-equiv="refresh" content="...">

Refreshes the page or redirects to another URL after a specified time.

Syntax:

<meta http-equiv="refresh" content="5; url=https://example.com">

Example:

<meta http-equiv="refresh" content="30">

<meta name="robots" content="...">

Controls how search engines index and follow links on the page.

Syntax:

<meta name="robots" content="index, follow">

Example:

<meta name="robots" content="noindex, nofollow">

<meta http-equiv="content-language" content="...">

Specifies the language of the webpage.

Syntax:

<meta http-equiv="content-language" content="en">

Example:

<meta http-equiv="content-language" content="en">

<meta property="og:..." content="...">

Provides metadata for social media platforms like Facebook and Twitter.

Syntax:

<meta property="og:title" content="Page Title">

Example:

<meta property="og:title" content="Metadata in HTML"> <meta property="og:description" content="Learn about metadata in HTML."> <meta property="og:image" content="https://example.com/image.jpg">

<link rel="icon" href="...">

Specifies a favicon for the webpage.

Syntax:

<link rel="icon" href="favicon.ico">

Example:

<link rel="icon" href="https://example.com/favicon.ico">

3.Text Content in HTML

<h1> to <h6>

Headings define the structure of the content. <h1> is the most important, and <h6> is the least important.

Syntax:

<h1>Heading 1</h1>

Example:

<h1>Main Heading</h1> <h2>Subheading</h2> <h3>Sub-subheading</h3>

<p>

Defines a paragraph of text.

Syntax:

<p>This is a paragraph.</p>

Example:

<p>This is a sample paragraph.</p>

<br>

Inserts a line break within text.

Syntax:

<br>

Example:

This is the first line.<br>This is the second line.

<hr>

Creates a horizontal line to separate content.

Syntax:

<hr>

Example:

<p>First section</p> <hr> <p>Second section</p>

<strong> or <b>

Makes text bold. <strong> is used for important text, while <b> is used for stylistic bold text.

Syntax:

<strong>Important Text</strong>

Example:

<p>This is <strong>important</strong> text.</p>

<em> or <i>

Makes text italic. <em> is used for emphasized text, while <i> is used for stylistic italic text.

Syntax:

<em>Emphasized Text</em>

Example:

<p>This is <em>emphasized</em> text.</p>

<u>

Underlines text.

Syntax:

<u>Underlined Text</u>

Example:

<p>This is <u>underlined</u> text.</p>

<s> or <del>

Strikes through text. <s> is used for text that is no longer relevant, while <del> is used for deleted text.

Syntax:

<s>Strikethrough Text</s>

Example:

<p>This is <s>strikethrough</s> text.</p>

<sup> and <sub>

<sup> is used for superscript text, and <sub> is used for subscript text.

Syntax:

<sup>Superscript</sup> and <sub>Subscript</sub>

Example:

<p>E = mc<sup>2</sup></p> <p>H<sub>2</sub>O is water.</p>

<pre>

Displays text exactly as written, preserving spaces and line breaks.

Syntax:

<pre>Preformatted Text</pre>

Example:

<pre> This is preformatted text. It preserves spaces and line breaks. </pre>

4. Lists

<ul>

Defines an unordered (bulleted) list.

Syntax:

<ul> <li>Item 1</li> <li>Item 2</li> </ul>

Example:

<ul> <li>Apple</li> <li>Banana</li> <li>Orange</li> </ul>

Output:

  • Apple
  • Banana
  • Orange

<ol>

Defines an ordered (numbered) list.

Syntax:

<ol> <li>Item 1</li> <li>Item 2</li> </ol>

Example:

<ol> <li>First</li> <li>Second</li> <li>Third</li> </ol>

Output:

  1. First
  2. Second
  3. Third

<li>

Defines a list item within <ul> or <ol>.

Syntax:

<li>List Item</li>

Example:

<ul> <li>Item 1</li> <li>Item 2</li> </ul>

Output:

  • Item 1
  • Item 2

<dl>

Defines a description list.

Syntax:

<dl> <dt>Term</dt> <dd>Definition</dd> </dl>

Example:

<dl> <dt>HTML</dt> <dd>HyperText Markup Language</dd> <dt>CSS</dt> <dd>Cascading Style Sheets</dd> </dl>

Output:

HTML
HyperText Markup Language
CSS
Cascading Style Sheets

<dt>

Defines a term in a description list.

Syntax:

<dt>Term</dt>

Example:

<dl> <dt>HTML</dt> <dd>HyperText Markup Language</dd> </dl>

Output:

HTML
HyperText Markup Language

<dd>

Defines a description for a term in a description list.

Syntax:

<dd>Definition</dd>

Example:

<dl> <dt>HTML</dt> <dd>HyperText Markup Language</dd> </dl>

Output:

HTML
HyperText Markup Language

5. Tables

<table>

Defines a table.

Syntax:

<table> <tr> <th>Header</th> <td>Data</td> </tr> </table>

Example:

<table> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>25</td> </tr> </table>

Output:

Name Age
John 25

<tr>

Defines a row in a table.

Syntax:

<tr> <th>Header</th> <td>Data</td> </tr>

Example:

<tr> <th>Name</th> <th>Age</th> </tr>

Output:

Name Age

<th>

Defines a header cell in a table.

Syntax:

<th>Header</th>

Example:

<th>Name</th>

Output:

Name

<td>

Defines a standard data cell in a table.

Syntax:

<td>Data</td>

Example:

<td>John</td>

Output:

John

<caption>

Defines a caption for a table.

Syntax:

<caption>Table Caption</caption>

Example:

<table> <caption>Employee Details</caption> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>25</td> </tr> </table>

Output:

Employee Details
Name Age
John 25

<colgroup>

Groups columns in a table for styling.

Syntax:

<colgroup> <col span="2" style="background-color:yellow"> </colgroup>

Example:

<table> <colgroup> <col span="1" style="background-color:yellow"> <col style="background-color:lightblue"> </colgroup> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>25</td> </tr> </table>

Output:

Name Age
John 25

<col>

Defines column properties within a <colgroup>.

Syntax:

<col style="background-color:yellow">

Example:

<col style="background-color:yellow">

Output:

Name
John

<thead>

Groups header content in a table.

Syntax:

<thead> <tr> <th>Header</th> </tr> </thead>

Example:

<table> <thead> <tr> <th>Name</th> <th>Age</th> </tr> </thead> <tbody> <tr> <td>John</td> <td>25</td> </tr> </tbody> </table>

Output:

Name Age
John 25

<tbody>

Groups body content in a table.

Syntax:

<tbody> <tr> <td>Data</td> </tr> </tbody>

Example:

<tbody> <tr> <td>John</td> <td>25</td> </tr> </tbody>

Output:

John 25

<tfoot>

Groups footer content in a table.

Syntax:

<tfoot> <tr> <td>Footer</td> </tr> </tfoot>

Example:

<table> <thead> <tr> <th>Name</th> <th>Age</th> </tr> </thead> <tbody> <tr> <td>John</td> <td>25</td> </tr> </tbody> <tfoot> <tr> <td>Total</td> <td>1</td> </tr> </tfoot> </table>

Output:

Name Age
John 25
Total 1

6. Forms

<form>

Defines a form for user input.

Syntax:

<form action="/submit" method="post"> <!-- Form elements go here --> </form>

Example:

<form action="/submit" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <button type="submit">Submit</button> </form>

Output:

<input>

Defines an input field.

Syntax:

<input type="text" name="field">

Example:

<input type="text" placeholder="Enter your name">

Output:

<textarea>

Defines a multi-line text input field.

Syntax:

<textarea rows="4" cols="50">Default text</textarea>

Example:

<textarea rows="4" cols="50" placeholder="Enter your message"></textarea>

Output:

<label>

Defines a label for a form element.

Syntax:

<label for="inputId">Label Text</label>

Example:

<label for="name">Name:</label> <input type="text" id="name" name="name">

Output:

<select>

Defines a dropdown list.

Syntax:

<select> <option value="1">Option 1</option> <option value="2">Option 2</option> </select>

Example:

<select> <option value="1">Option 1</option> <option value="2">Option 2</option> </select>

Output:

<option>

Defines an option in a dropdown list.

Syntax:

<option value="1">Option 1</option>

Example:

<option value="1">Option 1</option>

Output:

<optgroup>

Groups related options in a dropdown list.

Syntax:

<optgroup label="Group 1"> <option value="1">Option 1</option> <option value="2">Option 2</option> </optgroup>

Example:

<select> <optgroup label="Group 1"> <option value="1">Option 1</option> <option value="2">Option 2</option> </optgroup> </select>

Output:

<fieldset>

Groups related form elements.

Syntax:

<fieldset> <legend>Legend</legend> <!-- Form elements go here --> </fieldset>

Example:

<fieldset> <legend>Personal Information</legend> <label for="name">Name:</label> <input type="text" id="name" name="name"> </fieldset>

Output:

Personal Information

<legend>

Defines a caption for a <fieldset>.

Syntax:

<legend>Legend Text</legend>

Example:

<legend>Personal Information</legend>

Output:

Personal Information

<button>

Defines a clickable button.

Syntax:

<button type="button">Click Me</button>

Example:

<button type="button">Click Me</button>

Output:

<datalist>

Provides a list of predefined options for an <input>.

Syntax:

<input list="options"> <datalist id="options"> <option value="Option 1"> <option value="Option 2"> </datalist>

Example:

<input list="options" placeholder="Choose an option"> <datalist id="options"> <option value="Option 1"> <option value="Option 2"> </datalist>

Output:

<output>

Displays the result of a calculation.

Syntax:

<output name="result">0</output>

Example:

<output name="result">0</output>

Output:

0

<progress>

Represents the progress of a task.

Syntax:

<progress value="50" max="100">50%</progress>

Example:

<progress value="50" max="100">50%</progress>

Output:

50%

<meter>

Represents a scalar measurement within a known range.

Syntax:

<meter value="3" min="0" max="10">3/10</meter>

Example:

<meter value="3" min="0" max="10">3/10</meter>

Output:

3/10

7. Media Elements

<img>

Embeds an image in the document.

Syntax:

<img src="image.jpg" alt="Description">

Example:

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

Output:

<audio>

Embeds an audio file in the document.

Syntax:

<audio controls> <source src="audio.mp3" type="audio/mpeg"> Your browser does not support the audio element. </audio>

Example:

<audio controls> <source src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3" type="audio/mpeg"> Your browser does not support the audio element. </audio>

Output:

<video>

Embeds a video file in the document.

Syntax:

<video controls width="300"> <source src="video.mp4" type="video/mp4"> Your browser does not support the video element. </video>

Example:

<video controls width="300"> <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4"> Your browser does not support the video element. </video>

Output:

<source>

Specifies multiple media sources for <audio> or <video>.

Syntax:

<source src="media.mp4" type="video/mp4">

Example:

<video controls width="300"> <source src="video.mp4" type="video/mp4"> <source src="video.ogg" type="video/ogg"> Your browser does not support the video element. </video>

Output:

<track>

Adds subtitles or captions to <audio> or <video>.

Syntax:

<track src="subtitles.vtt" kind="subtitles" srclang="en" label="English">

Example:

<video controls width="300"> <source src="video.mp4" type="video/mp4"> <track src="subtitles.vtt" kind="subtitles" srclang="en" label="English"> Your browser does not support the video element. </video>

Output:

<figure>

Groups media content with a caption.

Syntax:

<figure> <img src="image.jpg" alt="Description"> <figcaption>Caption for the image</figcaption> </figure>

Example:

<figure> <img src="https://via.placeholder.com/150" alt="Placeholder Image"> <figcaption>This is a placeholder image.</figcaption> </figure>

Output:

<figcaption>

Defines a caption for a <figure>.

Syntax:

<figcaption>Caption Text</figcaption>

Example:

<figcaption>This is a caption.</figcaption>

Output:

8. Embedded Content

<iframe>

Embeds another webpage within the current document.

Syntax:

<iframe src="URL" width="300" height="200"></iframe>

Example:

<iframe src="https://www.example.com" width="100%" height="300"></iframe>

Output:

<embed>

Embeds external content, such as a PDF or Flash file.

Syntax:

<embed src="file.pdf" type="application/pdf" width="100%" height="300">

Example:

<embed src="https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf" type="application/pdf" width="100%" height="300">

Output:

<object>

Embeds multimedia, such as images, videos, or other webpages.

Syntax:

<object data="file.swf" type="application/x-shockwave-flash" width="100%" height="300"> Your browser does not support the object tag. </object>

Example:

<object data="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4" width="100%" height="300"> Your browser does not support the object tag. </object>

Output:

<param>

Defines parameters for an <object> element.

Syntax:

<param name="autoplay" value="true">

Example:

<object data="video.mp4" type="video/mp4" width="100%" height="300"> <param name="autoplay" value="true"> Your browser does not support the object tag. </object>

Output:

9. Semantic Tags

<header>

Represents the header of a document or a section.

Syntax:

<header> <h1>Page Title</h1> <p>Tagline or description</p> </header>

Example:

<header> <h1>Welcome to My Website</h1> <p>Your go-to place for HTML references</p> </header>

Output:

Welcome to My Website

Your go-to place for HTML references

<footer>

Represents the footer of a document or a section.

Syntax:

<footer> <p>Copyright © 2023</p> </footer>

Example:

<footer> <p>Copyright © 2023 My Website</p> </footer>

Output:

Copyright © 2023 My Website

<article>

Represents a self-contained piece of content, such as a blog post or news article.

Syntax:

<article> <h2>Article Title</h2> <p>Article content...</p> </article>

Example:

<article> <h2>Understanding Semantic HTML</h2> <p>Semantic HTML improves accessibility and SEO...</p> </article>

Output:

Understanding Semantic HTML

Semantic HTML improves accessibility and SEO...

<section>

Represents a thematic grouping of content.

Syntax:

<section> <h2>Section Title</h2> <p>Section content...</p> </section>

Example:

<section> <h2>Introduction</h2> <p>This is the introduction section...</p> </section>

Output:

Introduction

This is the introduction section...

<nav>

Represents a section of navigation links.

Syntax:

<nav> <a href="#">Home</a> <a href="#">About</a> <a href="#">Contact</a> </nav>

Example:

<nav> <a href="#">Home</a> <a href="#">About</a> <a href="#">Contact</a> </nav>

Output:

Home About Contact

<aside>

Represents content that is tangentially related to the main content, such as a sidebar.

Syntax:

<aside> <p>This is a sidebar.</p> </aside>

Example:

<aside> <p>This is a sidebar.</p> </aside>

Output:

This is a sidebar.

<main>

Represents the main content of the document.

Syntax:

<main> <h1>Main Content</h1> <p>This is the main content of the page.</p> </main>

Example:

<main> <h1>Main Content</h1> <p>This is the main content of the page.</p> </main>

Output:

Main Content

This is the main content of the page.
