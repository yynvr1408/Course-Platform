Introduction to JavaScript

What is JavaScript?

JavaScript is a powerful, high-level programming language used to make web pages interactive and dynamic. It works alongside HTML and CSS to add behavior like animations, form validation, content updates, and much more.

Why Use JavaScript?

JavaScript adds life to static websites. It allows you to:

  • Interact with the user through buttons, forms, and mouse events.
  • Manipulate and update HTML and CSS in real time.
  • Communicate with servers using APIs (AJAX or Fetch).
  • Create games, animations, and dynamic web apps.
  • Build full-stack applications using modern frameworks (e.g., Node.js, React).

Basic JavaScript Syntax

JavaScript code can be written inside a <script> tag or in a separate file with a .js extension.

Example (Inline Script):

<script>
  alert("Hello, JavaScript!");
</script>

How to Use JavaScript

There are three main ways to use JavaScript:

  1. Inline: Inside HTML elements using onclick, onchange, etc.
  2. Internal: Inside a <script> tag in the HTML file.
  3. External: In a separate JavaScript file linked to the HTML using <script src="script.js">.

Basic JavaScript Example

This example displays a greeting message in the browser console:

<script>
  function greet(name) {
    console.log("Hello, " + name + "!");
  }
  greet("Alice");
</script>

Common JavaScript Features

  • Variables: Store data using let, const, or var.
  • Data Types: String, Number, Boolean, Object, Array, Null, Undefined, Symbol.
  • Functions: Reusable blocks of code.
  • Events: Trigger code using clicks, hovers, inputs, etc.
  • DOM Manipulation: Change HTML/CSS dynamically with document methods.
  • Control Flow: Use if, for, while, etc.
  • APIs: Use fetch() to get data from servers.

Example (External Script)

Create a file called script.js:

function showMessage() {
  alert("JavaScript is connected!");
}

Then link it in your HTML:

<script src="script.js"></script>

JavaScript Variables

Variables in JavaScript are containers used to store data values. They are fundamental building blocks of any program and are used to label and reference data.

1. What Are Variables?

  • Used to store data like numbers, text, arrays, objects, etc.
  • Allow reuse and manipulation of values in a program.
  • Can be declared using var, let, or const.

2. Declaring Variables

Different keywords define the scope and behavior of variables.

// ES6 (modern JavaScript) let name = "Alice"; const pi = 3.1416; var count = 10; // old style, not recommended

3. let

let is used to declare block-scoped variables that can be reassigned.

let score = 50; score = 75; console.log(score); // 75

4. const

const declares block-scoped variables that cannot be reassigned.

const website = "example.com"; // website = "test.com"; // Error: Assignment to constant variable

5. var

var is function-scoped and hoisted. It's less predictable and not recommended in modern code.

var language = "JavaScript"; language = "TypeScript"; console.log(language);

6. Common Data Types

  • String: Text like "Hello"
  • Number: Numeric values like 42
  • Boolean: true or false
  • Array: Lists like [1, 2, 3]
  • Object: Key-value pairs like { name: "John" }
  • null and undefined

7. Common Use Cases

  • Storing user inputs and configuration values.
  • Calculating and storing results dynamically.
  • Manipulating DOM elements and user interactions.

8. Key Notes

  • Use let and const instead of var in modern JavaScript.
  • const is preferred for variables that don‚Äôt change.
  • Variables should have clear, descriptive names.
  • Always initialize variables when declaring them.

JavaScript Conditional Statements

Conditional statements in JavaScript are used to perform different actions based on different conditions. They allow you to add decision-making capabilities to your code.

1. What Are Conditional Statements?

  • Used to execute different code blocks based on specific conditions.
  • Types include: if, if...else, else if, switch, and ternary ?:.
  • Helps control program flow logically.

2. if Statement

Executes a block of code only if a specified condition is true.

let age = 20; if (age >= 18) { console.log("You are an adult."); }

3. if...else Statement

Executes one block if the condition is true, and another if it is false.

let isSunny = false; if (isSunny) { console.log("Go for a walk."); } else { console.log("Stay indoors."); }

4. if...else if...else Statement

Allows multiple conditions to be checked in sequence.

let grade = 75; if (grade >= 90) { console.log("Grade: A"); } else if (grade >= 80) { console.log("Grade: B"); } else if (grade >= 70) { console.log("Grade: C"); } else { console.log("Grade: D or below"); }

5. switch Statement

Performs different actions based on different values of a variable.

let day = "Sunday"; switch(day) { case "Monday": console.log("Start of the week."); break; case "Friday": console.log("End of the work week."); break; case "Sunday": console.log("Relax, it's the weekend!"); break; default: console.log("Just another day."); }

6. Ternary Operator

Short-form version of if...else for simple conditions.

let loggedIn = true; let message = loggedIn ? "Welcome back!" : "Please log in."; console.log(message);

7. Common Use Cases

  • Checking user input and taking actions accordingly.
  • Determining program flow based on environment or flags.
  • Handling different logic paths (grading, pricing tiers, etc.).

8. Key Notes

  • Always use === for strict equality in conditions.
  • Use else if chains when multiple checks are needed.
  • switch is cleaner for multiple discrete values.
  • Ternary is best used for small, inline conditions.

JavaScript Loops

Loops in JavaScript are used to run a block of code multiple times. They help reduce code repetition and automate repetitive tasks.

1. What Are Loops?

  • Allow code to be executed repeatedly based on a condition.
  • Main types: for, while, do...while, for...of, for...in.
  • Useful for iterating arrays, objects, or ranges of numbers.

2. for Loop

Executes a block of code a specific number of times.

for (let i = 1; i <= 5; i++) { console.log("Iteration " + i); }

3. while Loop

Executes as long as the condition is true.

let count = 1; while (count <= 3) { console.log("Count is: " + count); count++; }

4. do...while Loop

Runs the code block at least once, and then repeats while the condition is true.

let num = 1; do { console.log("Number is " + num); num++; } while (num <= 2);

5. for...of Loop

Used to iterate over iterable objects like arrays and strings.

const colors = ["red", "green", "blue"]; for (let color of colors) { console.log(color); }

6. for...in Loop

Used to iterate over the keys (properties) of an object.

const person = { name: "Alice", age: 25 }; for (let key in person) { console.log(key + ": " + person[key]); }

7. forEach Loop

Used to execute a provided function once for each array element. It‚Äôs a method available on array objects.

const fruits = ["apple", "banana", "cherry"]; fruits.forEach(function(fruit, index) { console.log(index + ": " + fruit); });

8. Key Notes

  • Always ensure loops have an exit condition to avoid infinite loops.
  • for...of is best for arrays, for...in for objects.
  • Use break to exit a loop early, and continue to skip to the next iteration.

JavaScript Array Methods

JavaScript provides many built-in methods to work with arrays. These methods help you add, remove, find, and transform elements efficiently.

1. What Are Array Methods?

  • Predefined functions to manipulate array content.
  • Help with searching, sorting, updating, and filtering arrays.
  • Work directly on array instances (e.g., arr.push()).

2. push()

Adds one or more elements to the end of an array.

const fruits = ["apple", "banana"]; fruits.push("cherry"); console.log(fruits); // ["apple", "banana", "cherry"]

3. pop()

Removes the last element from an array and returns it.

const fruits = ["apple", "banana", "cherry"]; fruits.pop(); console.log(fruits); // ["apple", "banana"]

4. shift()

Removes the first element from an array and returns it.

const fruits = ["apple", "banana"]; fruits.shift(); console.log(fruits); // ["banana"]

5. unshift()

Adds one or more elements to the beginning of an array.

const fruits = ["banana"]; fruits.unshift("apple"); console.log(fruits); // ["apple", "banana"]

6. map()

Creates a new array by applying a function to each element.

const nums = [1, 2, 3]; const doubled = nums.map(num => num * 2); console.log(doubled); // [2, 4, 6]

7. filter()

Creates a new array with only elements that pass a condition.

const nums = [1, 2, 3, 4]; const even = nums.filter(n => n % 2 === 0); console.log(even); // [2, 4]

8. find()

Returns the first element that satisfies a condition.

const nums = [3, 6, 9]; const found = nums.find(n => n > 5); console.log(found); // 6

9. forEach()

Executes a function for each array element.

const names = ["Alice", "Bob", "Eve"]; names.forEach(name => console.log(name));

10. includes()

Checks if an array contains a specific value.

const nums = [10, 20, 30]; console.log(nums.includes(20)); // true

11. slice()

Returns a shallow copy of a portion of an array.

const fruits = ["apple", "banana", "cherry"]; const sliced = fruits.slice(1, 3); console.log(sliced); // ["banana", "cherry"]

12. splice()

Adds or removes elements from an array.

const fruits = ["apple", "banana", "cherry"]; fruits.splice(1, 1, "blueberry"); console.log(fruits); // ["apple", "blueberry", "cherry"]

13. sort()

Sorts array elements alphabetically or by custom logic.

const nums = [4, 2, 9]; nums.sort((a, b) => a - b); console.log(nums); // [2, 4, 9]

14. Summary

  • push(), pop(), shift(), unshift() ‚Äî Add/Remove elements.
  • map(), filter(), forEach(), find() ‚Äî Iterate and transform.
  • slice(), splice() ‚Äî Extract or modify arrays.
  • includes(), sort() ‚Äî Check or organize array content.

JavaScript String Methods

String methods allow you to work with and manipulate text data in JavaScript. You can find, extract, change, or check strings using these built-in methods.

1. What Are String Methods?

  • Functions that operate on string values.
  • Help with searching, replacing, trimming, and formatting text.
  • Strings are immutable, so methods return new strings.

2. length

Returns the number of characters in a string.

const name = "Alice"; console.log(name.length); // 5

3. toUpperCase()

Converts all characters in a string to uppercase.

const text = "hello"; console.log(text.toUpperCase()); // "HELLO"

4. toLowerCase()

Converts all characters in a string to lowercase.

const text = "HeLLo"; console.log(text.toLowerCase()); // "hello"

5. includes()

Checks if the string contains a specified value.

const message = "JavaScript is awesome"; console.log(message.includes("awesome")); // true

6. indexOf()

Returns the index of the first occurrence of a value.

const text = "hello world"; console.log(text.indexOf("world")); // 6

7. slice()

Extracts a section of the string and returns it as a new string.

const str = "JavaScript"; console.log(str.slice(4, 10)); // "Script"

8. substring()

Similar to slice() but doesn't accept negative indices.

const str = "Programming"; console.log(str.substring(0, 6)); // "Progra"

9. replace()

Replaces a substring with another value.

const sentence = "I love cats"; console.log(sentence.replace("cats", "dogs")); // "I love dogs"

10. trim()

Removes whitespace from both ends of a string.

const text = " spaced out "; console.log(text.trim()); // "spaced out"

11. split()

Splits a string into an array based on a delimiter.

const data = "red,green,blue"; console.log(data.split(",")); // ["red", "green", "blue"]

12. charAt()

Returns the character at a specified index.

const word = "hello"; console.log(word.charAt(1)); // "e"

13. repeat()

Repeats a string a specified number of times.

const star = "*"; console.log(star.repeat(5)); // "*"

14. startsWith() & endsWith()

Check if a string starts or ends with a specified substring.

const phrase = "JavaScript"; console.log(phrase.startsWith("Java")); // true console.log(phrase.endsWith("Script")); // true

15. Summary

  • length, charAt(), slice() ‚Äî for accessing characters.
  • toUpperCase(), replace(), trim() ‚Äî for text cleanup.
  • includes(), indexOf(), startsWith() ‚Äî for searching.
  • split(), repeat() ‚Äî for transforming and formatting.

JavaScript provides built-in methods and the Math object for performing numeric operations like rounding, generating random numbers, and more.

1. What Are Number & Math Methods?

  • Number methods work directly on numeric values.
  • Math is a built-in object for advanced mathematical tasks.
  • Helps with rounding, power, random numbers, and constants.

2. toFixed()

Rounds a number down to the nearest integer.

console.log(Math.floor(4.9)); // 4

Rounds a number up to the nearest integer.

console.log(Math.ceil(4.1)); // 5

Returns a random number between 0 (inclusive) and 1 (exclusive).

11. Math.sqrt()

14. Summary

  • toFixed(), parseInt(), parseFloat() ‚Äî format and convert numbers.

  • Math.min(), max(), abs() ‚Äî comparison and absolute values.

JavaScript Object Methods

Objects in JavaScript are collections of key-value pairs. Object methods are used to manipulate, inspect, or extract information from objects.

1. What Are Object Methods?

  • Built-in methods to interact with JavaScript objects.
  • Helps in accessing, modifying, copying, or transforming object data.
  • Works with both simple and nested objects.

2. Object.keys()

Returns an array of an object's own property names (keys).

const user = { name: "Alice", age: 25 }; console.log(Object.keys(user)); // ["name", "age"]

3. Object.values()

Returns an array of an object's values.

const user = { name: "Alice", age: 25 }; console.log(Object.values(user)); // ["Alice", 25]

4. Object.entries()

Returns an array of key-value pairs from the object.

const user = { name: "Alice", age: 25 }; console.log(Object.entries(user)); // [["name", "Alice"], ["age", 25]]

5. Object.assign()

Copies values from one or more source objects to a target object.

const target = { a: 1 }; const source = { b: 2 }; const result = Object.assign(target, source); console.log(result); // { a: 1, b: 2 }

6. Object.freeze()

Makes an object immutable (no changes allowed).

const obj = { a: 1 }; Object.freeze(obj); obj.a = 2; console.log(obj.a); // 1 (change is ignored)

7. Object.seal()

Prevents new properties from being added, but allows changes to existing properties.

const obj = { a: 1 }; Object.seal(obj); obj.a = 2; obj.b = 3; // Won't be added console.log(obj); // { a: 2 }

8. hasOwnProperty()

Checks if a property exists in the object (not inherited).

const user = { name: "Alice" }; console.log(user.hasOwnProperty("name")); // true console.log(user.hasOwnProperty("age")); // false

9. Object.create()

Creates a new object using an existing object as a prototype.

const person = { greet() { console.log("Hi!"); } }; const student = Object.create(person); student.greet(); // Hi!

10. Summary

  • Object.keys(), values(), entries() ‚Äî convert object to array formats.
  • Object.assign() ‚Äî copy and merge objects.
  • Object.freeze(), seal() ‚Äî control mutability.
  • hasOwnProperty() ‚Äî verify ownership of a key.

JavaScript Date Methods

The Date object in JavaScript is used to work with dates and times. It allows you to get, set, and format date/time values in various ways.

1. Creating Dates

  • new Date() ‚Äî current date and time.
  • new Date("YYYY-MM-DD") ‚Äî specific date.
  • new Date(year, month, day, hours, minutes) ‚Äî individual parts.
const now = new Date(); const birthday = new Date("2000-01-01"); const custom = new Date(2023, 3, 5, 14, 30); // April 5, 2023

2. Getting Date Parts

  • getFullYear() ‚Äî Gets the 4-digit year.
  • getMonth() ‚Äî Gets month (0‚Äì11).
  • getDate() ‚Äî Gets day of the month.
  • getDay() ‚Äî Gets day of the week (0‚Äì6).
  • getHours(), getMinutes(), getSeconds()
const date = new Date(); console.log(date.getFullYear()); // e.g. 2025 console.log(date.getMonth()); // 0 = January console.log(date.getDate()); // 1‚Äì31 console.log(date.getDay()); // 0 = Sunday

3. Setting Date Parts

  • setFullYear(), setMonth(), setDate()
  • These methods modify the date object in place.
const d = new Date(); d.setFullYear(2030); d.setMonth(11); // December (months start at 0) console.log(d);

4. Formatting Dates

  • toDateString() ‚Äî returns date part only.
  • toISOString() ‚Äî returns ISO format string.
  • toLocaleDateString() ‚Äî localized string.
const today = new Date(); console.log(today.toDateString()); // "Sat Apr 05 2025" console.log(today.toISOString()); // "2025-04-05T10:20:30.000Z" console.log(today.toLocaleDateString()); // "4/5/2025" or similar

5. Comparing Dates

Dates can be compared by converting them to numbers using getTime().

const d1 = new Date("2025-01-01"); const d2 = new Date("2025-12-31"); console.log(d1.getTime() < d2.getTime()); // true

6. Calculating Time Difference

Subtract two date objects to get the difference in milliseconds.

const start = new Date("2025-01-01"); const end = new Date("2025-01-10"); const diffInMs = end - start; const diffInDays = diffInMs / (1000 * 60 * 60 * 24); console.log(diffInDays); // 9

7. Summary

  • Create: new Date(), new Date("YYYY-MM-DD")
  • Get: getFullYear(), getMonth(), etc.
  • Set: setDate(), setMonth(), etc.
  • Format: toDateString(), toLocaleDateString()
  • Compare: using getTime()

JavaScript Functions

Functions are blocks of code that are defined once and can be executed (called) whenever needed. They help in structuring code, improving readability, and reusability.

1. What is a Function?

  • A function is a reusable block of code designed to perform a specific task.
  • It can accept inputs (called parameters) and return an output using return.
  • Functions help avoid repetition and make programs modular and maintainable.

2. Function Syntax

function functionName(parameter1, parameter2) { // code to execute return result; }

3. Function Example

function greet(name) { return "Hello, " + name + "!"; } console.log(greet("Alice")); // Output: Hello, Alice!

4. Types of Functions

  • Function Declaration ‚Äì Standard way to declare a function using function.
  • Function Expression ‚Äì Function assigned to a variable.
  • Arrow Function ‚Äì Shorter syntax with =>, often used for simple logic.
  • IIFE (Immediately Invoked Function Expression) ‚Äì Runs immediately after creation.
// Function Declaration function sayHello() { return "Hello!"; } // Function Expression const sayHi = function() { return "Hi!"; }; // Arrow Function const sayHey = () => "Hey!"; // IIFE (function() { console.log("IIFE executed"); })();

5. Types of Parameters

  • Regular Parameters: Direct values passed during function call.
  • Default Parameters: Assigns a default value if no argument is given.
  • Rest Parameters: Gathers multiple arguments into a single array.
  • Destructured Parameters: Extracts data from objects/arrays passed into the function.
// Regular Parameter function add(a, b) { return a + b; } // Default Parameter function greet(name = "Guest") { return "Hello, " + name; } // Rest Parameter function sum(...numbers) { return numbers.reduce((a, b) => a + b); } // Destructured Parameter function printUser({ name, age }) { console.log(${name} is ${age} years old); }

6. Function Methods

  • call() ‚Äì Invokes function with a specified this value and arguments.
    function sayName() { console.log(this.name); }
    sayName.call({ name: "Sam" });
  • apply() ‚Äì Same as call(), but arguments are in an array.
    function introduce(age, city) { console.log(${this.name}, ${age}, from ${city}); }
    introduce.apply({ name: "John" }, [30, "Paris"]);
  • bind() ‚Äì Returns a new function with a fixed this value.
    const user = { name: "Eva" };
    function hello() { console.log("Hi " + this.name); }
    const boundHello = hello.bind(user);
    boundHello();
  • toString() ‚Äì Returns the function as a string.
    function show() { return "Done"; }
    console.log(show.toString());

7. Example (call, apply, bind)

function introduce(greeting) { console.log(greeting + ", I am " + this.name); } const user = { name: "Liam" }; introduce.call(user, "Hello"); // Hello, I am Liam introduce.apply(user, ["Hi"]); // Hi, I am Liam const boundIntro = introduce.bind(user); boundIntro("Hey"); // Hey, I am Liam

8. Key Notes

  • Use return to send data back from a function.
  • Arrow functions do not bind their own this.
  • Functions can be passed as arguments to other functions (callbacks).
  • Functions inside functions create closures, allowing persistent data.

JavaScript Async & Await

Async/Await simplifies working with asynchronous operations, making asynchronous code look and behave more like synchronous code.

1. What is Async/Await?

  • async keyword is used to declare a function that returns a promise.
  • await is used inside async functions to pause execution until a Promise is resolved or rejected.
  • Makes asynchronous code easier to read and manage than using plain Promises or callbacks.

2. Basic Example

async function getMessage() { return "Hello from async!"; } getMessage().then(msg => console.log(msg)); // Output: Hello from async!

3. Await Example

function fetchData() { return new Promise(resolve => { setTimeout(() => resolve("Data loaded"), 1000); }); } async function load() { const data = await fetchData(); console.log(data); } load(); // Output after 1 second: Data loaded

4. Error Handling

You can handle errors using try...catch blocks with async functions.

async function fetchData() { try { const res = await fetch('https://api.invalidurl.com/data'); const data = await res.json(); console.log(data); } catch (error) { console.error("Error:", error); } } fetchData();

5. Key Notes

  • await can only be used inside an async function.
  • Helps avoid deeply nested .then() chains.
  • Combines best of both worlds: readable syntax and asynchronous power.

JavaScript Callbacks

A callback is a function passed as an argument to another function, to be executed later. Callbacks are used to handle asynchronous tasks like reading files or making API requests.

Example:

function greet(name, callback) {

JavaScript Recursion

Recursion is a technique where a function calls itself to solve a smaller version of a problem, until it reaches a base case.

Example: Factorial

function factorial(n) {
  if (n === 0) return 1;
  return n * factorial(n - 1);
}

console.log(factorial(5)); // Output: 120

A Promise is an object that represents the eventual completion or failure of an asynchronous operation. Promises have three states: pending, fulfilled, and rejected.

Callback Hell

Callback Hell occurs when callbacks are nested within callbacks, leading to code that is hard to read and maintain. This often happens in asynchronous code without proper structure.

Example:

setTimeout(() => {
  console.log("Step 1");
  setTimeout(() => {
    console.log("Step 2");
    setTimeout(() => {
      console.log("Step 3");
    }, 1000);
  }, 1000);
}, 1000);

To avoid callback hell, use Promises or async/await for better readability and maintainability.

JavaScript map(), filter(), and reduce()

These powerful array methods help you transform, filter, and summarize data efficiently. They are widely used in functional programming patterns.

1. map()

map() creates a new array by applying a function to each element in the original array.

const numbers = [1, 2, 3, 4]; const doubled = numbers.map(num => num * 2); console.log(doubled); // [2, 4, 6, 8]

2. filter()

filter() returns a new array with elements that pass a test (return true).

const numbers = [1, 2, 3, 4, 5]; const evens = numbers.filter(num => num % 2 === 0); console.log(evens); // [2, 4]

3. reduce()

reduce() reduces an array to a single value by applying a function to each element (accumulator pattern).

const numbers = [1, 2, 3, 4]; const sum = numbers.reduce((total, num) => total + num, 0); console.log(sum); // 10

4. When to Use Each

  • map(): When you want to transform items in an array.
  • filter(): When you want only certain items from an array.
  • reduce(): When you want a single value (sum, average, object, etc.).

5. Real-World Combo Example

Combining all three methods:

const products = [ { name: "Pen", price: 10 }, { name: "Book", price: 50 }, { name: "Pencil", price: 5 } ]; const total = products .filter(p => p.price > 5) .map(p => p.price) .reduce((sum, price) => sum + price, 0); console.log(total); // 60

6. Key Notes

  • These methods do not change the original array.
  • map() and filter() return arrays, reduce() returns a single value.
  • Can be chained together for powerful data manipulation.

JavaScript fetch()

The fetch() method is used to make network requests (like APIs) and handle responses asynchronously. It returns a Promise and is often used with .then() or async/await.

1. Syntax

fetch(url, options) .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));

2. Basic GET Request

Fetch data from an API (read-only).

fetch('https://jsonplaceholder.typicode.com/posts/1') .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));

3. POST Request

Send data to a server using JSON format.

fetch('https://jsonplaceholder.typicode.com/posts', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ title: 'New Post', body: 'This is the content.', userId: 1 }) }) .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error('Error:', error));

4. Using async/await

Cleaner and modern way to handle asynchronous fetch logic.

async function getPost() { try { const response = await fetch('https://jsonplaceholder.typicode.com/posts/1'); const data = await response.json(); console.log(data); } catch (error) { console.error('Error:', error); } } getPost();

5. Common Use Cases

  • Fetching data from RESTful APIs.
  • Submitting form data to a server.
  • Loading dynamic content without page reload.
  • Interacting with external services (e.g., weather, search).

6. Key Notes

  • fetch() is Promise-based and works asynchronously.
  • Always handle errors using .catch() or try...catch.
  • Use response.json() to parse the response body into a JavaScript object.
  • To send data, include method, headers, and body in the options object.

JSON (JavaScript Object Notation)

JSON is a lightweight data format used for storing and exchanging data. It is easy to read, write, and parse ‚Äì making it ideal for APIs and communication between systems.

1. What is JSON?

  • Data format based on JavaScript object syntax.
  • Used to exchange data between server and client.
  • Supported by almost every modern programming language.

2. Example JSON

{ "name": "Alice", "age": 25, "isStudent": false, "skills": ["HTML", "CSS", "JavaScript"] }

3. JSON.stringify()

Converts a JavaScript object into a JSON string.

const obj = { name: "Bob", age: 30 }; const jsonString = JSON.stringify(obj); console.log(jsonString); // Output: {"name":"Bob","age":30}

4. JSON.parse()

Converts a JSON string into a JavaScript object.

const jsonText = '{"name":"Eve","age":28}'; const user = JSON.parse(jsonText); console.log(user.name); // Output: Eve

5. Common Use Cases

  • Sending and receiving data from APIs.
  • Storing settings or configurations.
  • Reading/writing structured data to files or localStorage.

6. Key Notes

  • JSON keys must be in double quotes.
  • Functions and undefined are not valid in JSON.
  • JSON.stringify() ignores non-serializable values.
  • JSON.parse() will throw an error for invalid JSON.
